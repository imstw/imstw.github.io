MongoDB 安裝

-安裝MongoDB 5.0

MongoDB Download
1.安裝過程暫不安裝 MongoDB Compass，要取消勾選。

2.安裝好後，建立儲存的資料夾:(cmd=> mkdir c:\data\db)
C:\data\db
C:\data\log

3.在MongoDB安裝目錄下建立檔案，檔名：mongod.cfg
內容:
systemLog:
destination: file
path: c:\data\log\mongod.log
storage:
dbPath: c:\data\db\

備註:
systemLog：path 為前步驟 log 資料夾位置
storage：dbPath 為前步驟 db 資料夾位置

4.建立服務:
(cmd=> “C:\Program Files\MongoDB\Server\5.0\bin\mongod.exe” –config “C:\Program Files\MongoDB\Server\5.0\mongod.cfg” –install)

工作管理員–>服務–>啟用MongoDB服務
