[VBA]Excel Connect To Oracle Database

By 葳 20 January 2023

Excel connect to Oracle Database

定義方法 ExceltoOracle_InsertData()
Sub ExceltoOracle_InsertData()
定義讀取的資料範圍
Dim RowCount As Long
RowCount = Worksheets("原始資料").Range("B" & Rows.Count).End(xlUp).Row
設定連線Oracle資料庫
Dim con As Object, com As Object, rec As Object
Set con = CreateObject("ADODB.Connection")
Set com = CreateObject("ADODB.Command")
Set rtTable = CreateObject("ADODB.Recordset")

'Open connection(ORACLE TNS 的設定)
con.Open "Provider=OraOLEDB.Oracle;User ID=TestDB;Data Source=127.0.0.1:1521/ivy;Password=1234"
OraOpen = True
連線成功跳出成功訊息
MsgBox "Connect to the oracle database Successful!", vbInformation, "Connect Successful"
把Excel表格資料寫入Oracle資料庫中的"新進員工資料表"
'Open Table and insert data
rtTable.Open "select * from 新進員工資料表", con, 3, 3
If RowCount > 0 Then
For i = 2 To RowCount
  rtTable.AddNew
  rtTable.Fields("員工編號").Value = ActiveSheet.Range("A" & i).Value
  rtTable.Fields("員工姓名").Value = ActiveSheet.Range("B" & i).Value
  rtTable.Fields("員工性別").Value = ActiveSheet.Range("C" & i).Value
  If ActiveSheet.Range("D" & i).Value = "" Then
    rtTable.Fields("入職日期").Value = DBNull
  Else
    rtTable.Fields("入職日期").Value = ActiveSheet.Range("D" & i).Value
  End If
Next
End If

rtTable.Update
資料寫入(更新)成功則會跳出成功訊息
MsgBox "COLUMN_1、COLUMN_2 Update Successful!", vbInformation, "Update Successful"
關閉與資料庫的連線
'Close Table and close connect
rtTable.Close
Set rtTable = Nothing
con.Close
Set con = Nothing
  
End Sub
